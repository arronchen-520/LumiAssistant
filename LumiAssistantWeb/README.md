# ✦ Lumi Assistant — Web

> 你的 AI 私人秘书，帮你记事、管待办、记住你的一切。
>
> **零后端、零安装、零追踪 — 一切数据都存在你的浏览器里。**

详细功能介绍请查看 [主 README](../README.md)。

## 🚀 使用方式

### 方式一：直接打开
双击 `index.html`，在浏览器里直接使用。

### 方式二：部署到 Netlify / Vercel
```bash
# Netlify CLI
netlify deploy --dir .

# 或直接拖拽文件夹到 netlify.com/drop
```

### 方式三：手机 PWA
1. 用手机浏览器访问部署后的 URL
2. 点击「添加到主屏幕」
3. 从桌面图标打开 = 全屏 App 体验
